# Contact Manager (Linked List in C++)

A console-based contact management system implemented using a singly linked list.

## Features
- Add contact
- View all contacts
- Search contact by name
- Delete contact

## Concepts Used
- Pointers
- Dynamic memory alloaction
- Linked list traversal
- Insert and delete operations

## How to Run
Compile and run using Dev-C++ or any GCC compiler.
